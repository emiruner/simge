Examples

The simge-examples package contains example programs utilizing Simge library. Every example is in its own directory. Here is a partial list of the examples:

  • atherton: Weiler-Atherton clipping algorithm demo
  • containmentcontrol: Point inside polygon test
  • giftwrap: Gift-wrap point enclosure algorithm
  • lineplane: Line-plane intersection test
  • mazesolver: Maze generation and maze solver demo
  • monotone: Monotone polygon triangulation
  • triangletriangle: Triangle-triangle intersection
